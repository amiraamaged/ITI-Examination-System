# Examination Management System – ITI Graduation Project (Power BI Track)

A complete end-to-end Examination Management System developed as part of the ITI Power BI Track Graduation Project, designed to automate and manage online examinations, student tracking, and performance analytics — simulating the real-world ITI ecosystem.
# ----
## Project Overview

The system manages exam creation, execution, grading, and performance analysis. It integrates a SQL Server backend, automated ETL processes, reporting tools, and a Flask web application to deliver a unified examination and analytics platform.
# ----
## Key Features

- Dynamic exam generation and automated grading
- Role-based student and instructor access
- Bulk answer submission using table-valued parameters
- Exam performance tracking through SSRS and Power BI
- Snowflake schema tailored for BI and reporting
- End-to-end data pipeline (Excel → SSIS → SQL → Flask → Power BI)
# ----
## Technologies Used

- SQL Server: Database design, stored procedures, triggers, indexing
- SSIS: ETL workflows for structured data loading
- SSRS: Analytical and operational reports
- Power BI: Semantic model, DAX measures, and 20+ interactive dashboards
- Flask Framework: Web application for exam delivery and management
- Excel & Faker library: Data creation and generation.
# ----
## Project Structure

- Database Design – ERD, logical model, and relationships
- Database Implementation – Tables, constraints, indexes, triggers, stored procedures
- Data Preparation – Sample data + Faker-generated datasets
- Reporting & Visualization – SSRS reports + Power BI dashboards
- Web Application – Flask-based exam portal

  ### Roles
  - Student: Take exams, view results and performance
  - Instructor: Create exams, monitor progress, access analytics.
